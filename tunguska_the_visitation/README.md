# Tunguska: The Visitation
Auto Splitter and Load Remover for [Tunguska: The Visitation](https://www.speedrun.com/tunguska_the_visitation). Based off [ash-help](https://github.com/just-ero/asl-help) library by [Ero](https://github.com/just-ero).
## Usage
* [Downolad](https://github.com/just-ero/asl-help/blob/main/lib/asl-help) asl-help library for LiveSplit.
* Copy `asl-help` from lib folder to LiveSplit Components folder.
* Download the script.
* For LiveSplit add Edit Layout... > Control > Scriptable Auto Splitter to your layout.
* Open the layout settings, and point it to the file you downloaded.
