# Tunguska: The Visitation
Auto Splitter and Load Remover for [Tunguska: The Visitation](https://www.speedrun.com/tunguska_the_visitation). Based on [ash-help](https://github.com/just-ero/asl-help) library by [Ero](https://github.com/just-ero) and [uhara](https://github.com/ru-mii/uhara) library by [rumii](https://github.com/ru-mii/uhara).
## Usage
* [Downolad](https://github.com/just-ero/asl-help/blob/main/lib/asl-help) asl-help library for LiveSplit.
* Place `asl-help` to LiveSplit Components folder.
* [Download](https://github.com/ru-mii/uhara/blob/main/bin/uhara9) uhara9 library for LiveSplit.
* Place `uhara9` to LiveSplit Components folder.
* Download the script.
* For LiveSplit add Edit Layout... > Control > Scriptable Auto Splitter to your layout.
* Open the layout settings, and point it to the file you downloaded.
